Definicion:

Las Aplicaciones Web Progresivas (PWA) son una nueva generación de aplicaciones que desdibujan las fronteras entre aplicaciones web y apps móviles tradicionales combinando las ventajas de ambas: mantienen todas las funciones de las apps móviles, pero solo necesitan un navegador web para ser utilizadas.
